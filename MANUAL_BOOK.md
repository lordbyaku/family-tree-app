### 4.1 Menambah Anggota Baru

#### Langkah-Langkah:

1. Klik tombol **"+ Add Member"** di toolbar

#### Tab 2: Informasi Hubungan

- **Pendidikan**

- **Nomor Telepon**
- **Email**
- **Catatan Tambahan** - Informasi lain yang ingin dicatat

3. Setelah semua terisi, klik **"Save Member"**
4. Anggota baru akan muncul di pohon keluarga

### 4.2 Mengedit Informasi Anggota

1. Klik pada **node/kartu anggota** di pohon keluarga
2. Modal detail anggota akan muncul
3. Klik tombol **"Edit"** (ikon pensil)

## 7. FITUR EKSPOR DATA

### 7.1 Jenis Ekspor yang Tersedia

Aplikasi mendukung 4 format ekspor:
1. **PDF** - Buku keluarga dalam format PDF
2. **Excel** - Data tabulasi dalam spreadsheet
3. **PNG** - Gambar pohon keluarga
4. **HTML** - Halaman web standalone

### 7.2 Ekspor ke PDF

**Langkah-Langkah:**

1. Klik tombol **"Export"** di toolbar
2. Pilih **"Export to PDF"**
3. Modal ekspor akan muncul dengan opsi:
   - **Family Book Style** - Buku keluarga lengkap dengan semua detail
   - **Tree Diagram** - Hanya diagram pohon
   - **Include Photos** - Sertakan foto anggota
   - **Include Statistics** - Sertakan statistik

4. Sesuaikan pengaturan sesuai kebutuhan
5. Klik **"Generate PDF"**
6. Tunggu proses generate (bisa memakan waktu untuk data besar)
7. File PDF akan otomatis terdownload

**Isi PDF Family Book:**
- Cover dengan nama keluarga
- Daftar isi
- Pohon keluarga (diagram)
- Profil detail setiap anggota
- Statistik keluarga
- Timeline sejarah keluarga

### 7.3 Ekspor ke Excel

**Langkah-Langkah:**

1. Klik tombol **"Export"** di toolbar
2. Pilih **"Export to Excel"**
3. File Excel akan langsung terdownload

**Isi File Excel:**

File Excel berisi beberapa sheet:
- **Members** - Data lengkap semua anggota
- **Relationships** - Relasi antar anggota
- **Statistics** - Ringkasan statistik
- **Birthdays** - Daftar ulang tahun

**Kolom di Sheet Members:**
- ID, Nama, Gender, Tanggal Lahir, Tempat Lahir

### 7.4 Ekspor ke PNG (Gambar)

**Langkah-Langkah:**

1. Atur tampilan pohon sesuai keinginan (zoom, posisi)

4. Pilih kualitas:

6. File gambar akan terdownload

**Tips:**
- Gunakan "Fit View" sebelum ekspor untuk capture seluruh pohon
- Untuk kualitas terbaik, pilih High Quality
- File PNG dapat digunakan untuk presentasi atau cetak

### 7.5 Ekspor ke HTML

**Langkah-Langkah:**

1. Klik tombol **"Export"** di toolbar
2. Pilih **"Export to HTML"**
3. File HTML akan terdownload

## 8. FITUR HUBUNGAN KELUARGA

### 8.1 Kalkulator Hubungan

Fitur ini menghitung tingkat kekerabatan antara dua anggota keluarga.

**Cara Menggunakan:**

1. Klik node **anggota pertama** di pohon
2. Di modal yang muncul, klik **"Calculate Relationship"**
3. Pilih **anggota kedua** dari dropdown
4. Hasil hubungan akan ditampilkan

**Contoh Hasil:**
- "Ayah" / "Ibu"
- "Anak laki-laki" / "Anak perempuan"
- "Kakek" / "Nenek"
- "Cucu"
- "Saudara kandung"
- "Sepupu"
- "Paman" / "Bibi"
- "Keponakan"
- "Saudara ipar"
- "Kakak ipar" / "Adik ipar"
- Dan lainnya...

### 8.2 Jalur Hubungan (Relationship Path)

**Cara Menggunakan:**

1. Klik tombol **"Relationship Path"** di modal anggota
2. Pilih anggota tujuan
3. Aplikasi akan menampilkan jalur lengkap:
   - Nama setiap anggota dalam jalur
   - Tipe hubungan di setiap langkah
   - Visual jalur dengan panah

## 13. TIPS DAN TRIK

### 13.1 Efisiensi Input Data

**Input Data Keluarga Besar:**

1. **Mulai dari Generasi Tertua**
   - Input kakek-nenek terlebih dahulu
   - Lalu anak-anaknya
   - Kemudian cucu, dan seterusnya

2. **Gunakan Template Excel**
   - Lebih cepat untuk data banyak
   - Bisa dikerjakan offline
   - Import sekaligus

3. **Isi Minimal Dulu**
   - Prioritas: Nama, gender, tanggal lahir
   - Detail lain bisa ditambahkan nanti

### 13.2 Organisasi Foto

**Best Practices:**

1. **Naming Convention**
   - Gunakan nama file yang jelas
